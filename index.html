How To Play
ANDROID (POWERTUNNEL)

Download PowerTunnel. Buka Settings > DNS Providers > Manual. Masukkan IP Host di daftar filter.

IOS (SURGE 5)

Gunakan Surge 5. Edit Config, tambahkan hostname lo ke arah IP server. Pastikan VPN aktif.

VIRTUAL HOST (NO ROOT)

Paling gampang! Klik tombol di bawah untuk login lewat portal browser otomatis.

WINDOWS PC

Edit file C:\Windows\System32\drivers\etc\hosts. Tambahkan: [IP] [Hostname] di baris paling bawah.
